Pemahaman Design Website

Apa itu adaptive and responsive web design?
Adaptive web design adalah design web yang akan secara design dan juga bisa content akan menyesuaikan jika user sedang menggunakan device tertentu (e.g. mobile). Sedangkan responsive web design adalah design web yang hanya mengubah dimensi atau sizing tergantung device apa yang user sedang gunakan

Apakah hadirnya smartphone dan tablet mengubah design web secara signifikan? Bagaimana kira-kira teknologi baru akan mengubah design web di masa depan?
Ya, itu telah mengubah design web secara signifikan. Karena penggunaan tablet, mobile, dan pc sangat berbeda dari segi user. Perbedaan utama ada pada: jarak pandang, cara menggunakan kursor, besarnya layar, bahkan koneksi internet. Jadi untuk setiap device, jika ingin kita maksimalkan penggunaan website kita, kita harus menyesuaikannya sesuai dengan cara user memakai website kita. Mengenai tekonologi baru, saya rasa design website akan lebih rumit lagi. Misalnya saja, jika wearable device (e.g. google glass) sudah secara umum digunakan untuk browsing, maka perlu untuk design website untuk disesuaikan.

Bagaimana kira-kira orang yang memiliki kekurangan fisik menggunakan website? Apakah kita perlu menyesuaikan website untuk mereka juga? Solusi apa yang dapat mengatasi keadaan tersebut?
Jawabannya adalah tergantung tujuan dari website tersebut. Jika tujuannya juga untuk menarik orang-orang yang memiliki kekurangan fisik, maka itu perlu dilakukan. Contoh kekurangan fisik yang saya pikirkan adalah kebutaan. Untuk mengatasinya, maka bisa menggunakan audio reader yang dapat membaca tulisan yang merupakan content, tulisan yang merupakan content, tulisan yang dianggap penting, dan sebagainya.

Ide atau konsep apa yang sesuai denganmu saat mempelajari desain web?
Saya sesuai dengan adaptive web design dibandingkan dengan responsive web design. Semakin kita bisa menyesuaikan fungsi website kita dengan apa yang user butuhkan, akan semakin baik
